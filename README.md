# シアレ株式会社 コーポレートサイト

マンション運営コンサルタント「シアレ株式会社」の Web サイト（静的サイト：HTML / CSS / JavaScript）。

## 構成

```
siare/
├─ index.html          … ページ本体（全セクション）
├─ css/style.css       … スタイル（デザインシステム＋全レイアウト）
├─ js/main.js          … メニュー開閉・ヘッダー切替
├─ images/             … 画像（hero-1〜4.jpg ほか）
├─ _server.pl          … プレビュー用の簡易ローカルサーバ（開発専用）
└─ .claude/launch.json … プレビュー設定
```

### セクション
ヒーロー（背景スライドショー）／ブランドコンセプト／サービス／向き合う管理組合／代表メッセージ／会社概要／お問い合わせフォーム

## 複数端末での作業手順

このリポジトリを GitHub 経由で同期します。**どの端末でも同じ GitHub アカウント**を使用してください。

### 別の端末で初めて使うとき
```bash
git clone <このリポジトリのURL>
cd siare
```

### 作業の開始時（最新を取得）
```bash
git pull
```

### 作業の終了時（変更を保存・共有）
```bash
git add -A
git commit -m "変更内容のメモ"
git push
```

> 複数端末で編集するときは、**作業前に必ず `git pull`、作業後に `git push`** を徹底すると競合を防げます。

## ローカルプレビュー

ブラウザで `index.html` を直接開いても表示できます。
スライドショー等を含めサーバ経由で確認したい場合は、Perl があれば次で起動できます。

```bash
perl _server.pl 4173
# → http://127.0.0.1:4173 を開く
```

Perl が無い環境では、Python など任意の静的サーバでも代用できます。
```bash
# 例: Python がある場合
python -m http.server 4173
```
